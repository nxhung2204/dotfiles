# dotfiles

Personal macOS development environment configuration managed via GNU Stow.

## Quick Setup

```bash
chmod +x install.sh
./install.sh
```

## What's Included

| Tool | Config path |
|------|-------------|
| Neovim | `.config/nvim/` |
| Tmux | `.config/tmux/` |
| Wezterm | `.config/wezterm/` |
| AeroSpace | `.config/aerospace/` |
| Borders | `.config/borders/` |
| Zsh | `.zshrc` |
| Git | `.gitconfig` |
| Starship | `.config/starship.toml` |
